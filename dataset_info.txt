XR-Bone Dataset Information

Datasets are NOT included in this repository due to size and licensing restrictions.

This project used multiple public X-ray datasets for classification and localization.

============================================================
1. Classification Datasets
============================================================

The final classification dataset was named:

It was created by combining multiple fracture / no-fracture datasets.

4. Notes
============================================================

The repository contains code, model loading logic, backend API, Flutter frontend, and model weights.

The raw datasets are not included. Users must download or prepare datasets separately if they want to retrain the models.
